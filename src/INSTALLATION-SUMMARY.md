## 🚀 How to Run (3 Steps)

### Step 1: Install Node.js
Download from: https://nodejs.org/

### Step 2: Install Dependencies
```bash
npm install
```

### Step 3: Start Development Server
```bash
npm run dev
```

🎉 **Done!** Visit http://localhost:3000

---

## 📋 Alternative Quick Start Methods

### Method 1: Using VS Code Terminal (Recommended)
1. Open project in VS Code
2. Press `Ctrl + ` ` to open terminal
3. Run: `npm install`
4. Run: `npm run dev`

### Method 2: Using Windows Batch File
1. Install dependencies first: `npm install`
2. Double-click **start-dev.bat** file
3. Browser opens automatically

### Method 3: Using Mac/Linux Shell Script
1. Install dependencies first: `npm install`
2. Make executable: `chmod +x start-dev.sh`
3. Run: `./start-dev.sh`
